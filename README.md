This is a Python-based web application built with Streamlit to upload and view sunset videos.

## Prerequisites

You need to have **Python** installed on your computer to run this application.

### 1. Install Python
If you don't have Python installed:
1.  Go to [python.org](https://www.python.org/downloads/).
2.  Download the latest version for Windows.
3.  **IMPORTANT**: When installing, check the box that says **"Add Python to PATH"**.

### 2. Install Dependencies
Open a terminal (Command Prompt or PowerShell) in this folder and run:
```bash
pip install -r requirements.txt
```

## How to Run
In the terminal, run:
```bash
streamlit run app.py
```

The app should open automatically in your browser.
